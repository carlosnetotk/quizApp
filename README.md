# Quiz App

Esse é um projeto simples construído com React/TypeScript e consumindo uma API de terceiro (https://opentdb.com/api_config.php)

### Observações

- Quiz App não é um projeto para mostrar conhecimento em segurança digital
- Algumas informações no repositório podem estar erradas pois esta é a minha primeira vez utilizando um gerenciador de versões. Ainda estou aprendendo :)

# Features

- O aplicativo irá consumir a API externa para gerar 10 perguntas sobre cartoons e animações.
- Cada pergunta terá 4 possíveis respostas.
- Ao clicar em uma possível resposta o usuário terá o feedback com alteração da cor dos botões e com uma pontuação.
- Ao iniciar o jogo as perguntas aparecerão e ao termino das 10 perguntas, aparecerá um botão para reiniciar o jogo.
